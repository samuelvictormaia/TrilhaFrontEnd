# 🧠 Plataforma EAD Local

Plataforma interativa desenvolvida para **organizar estudos de Front-End**, funcionando totalmente **offline**.  
Criada com HTML, CSS e JavaScript puro, simulando um ambiente de curso EAD.

---

## 🎯 Objetivo
Ajudar no aprendizado progressivo de:
- HTML, CSS, JavaScript  
- React, Tailwind, UX/UI e Scrum  
- Inglês técnico  

---

## 🚀 Tecnologias utilizadas
- HTML5  
- CSS3  
- JavaScript  
- LocalStorage (para salvar progresso)  

---

## 🧱 Funcionalidades
✅ Menu lateral com módulos  
✅ Progresso salvo localmente  
✅ Botão de “Marcar como concluído”  
✅ Interface limpa e responsiva  

---

## 🧩 Como executar
1. Baixe a pasta do projeto e extraia.  
2. Abra o arquivo `index.html` no navegador.  
3. Use normalmente, sem precisar de internet.
